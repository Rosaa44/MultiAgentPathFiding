Pour voir les différentes stratégies en action, dans le fichier main.py, des lignes 294 à 300, on peut décommenter la ligne qui indique la stratégie souhaitée.
Ex: on décommente la ligne 298 « strategie2(nbiterations) » pour voir la stratégie 2 en action.

Le fichier main.py permet de voir 2 équipes jouer avec la même stratégie, par exemple dans stratégie 2, les deux équipes utilisent l’algorithme du cooperative pathfinding.

Le fichier main2.py permet de voir 2 équipes jouer avec des stratégies différentes: Path slicing et A* avec recalcul du path.
